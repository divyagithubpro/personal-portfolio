ONLINE PSYCHIATRIST APPOINTMENT BOOKING

The project ONLINE PSYCHIATRIST APPOINTMENT BOOKING is a internet based psychiatrist appointment booking system which is helpful for patients in hospital . The aim of this project is to provide ease and comfort to depression patients while taking an appointment from psychiatrist and it also resolves the problems that the patients has to face while making an appointment.This project will reduce human effort like paper work ,time lose , lack of accuracy etc. It is an effective method because the information is stored in the database .All we need to do is open the database using internet and view the appointments or transfer the appointment status of each patient electronically using adequate devices instead of manually storing data.it can be used not only for taking appointment but also to know the various hospitals that have psychiatrists.

Software Requirements

  • Operating System : Windows / Linux etc
    • IDE :Visual studio code
  • Back End : SQLite 3
  • Front End : Python 3.7
  • Documentation : MS Office
  • Designing tools : HTML, JAVASCRIPT, CSS, jQuery
