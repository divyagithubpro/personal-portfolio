GARBOUT APP

Garbout is a mobile application for garbage management. The quantity and generation rate of waste will continuously increase tremendously as long as human activities continue to exist. The consequence caused by badly management of waste, it is very dangerous to the world and human health. This calls for the need to salvage the situation before it gets out of hand. That conducts us to use best practices to properly manage waste collection following the new trend of mobile technology, this research developed a waste collection management model with the aim to design a Mobile Application model for waste collection management which will help to improve the quality of management in Waste collection and impacting positively the environment by reducing the quantity of Solid waste .Android application that allows a user to manage waste efficiently using the 3 R's i.e. Reduce, Reuse and Recycle. It notifies all the volunteers and the Small Scale Industries who are in the close vicinity of the donor about all the information of the donation which facilitates easy and transparent collection of waste from door to door. we could now eradicate most issues in the existing system. some of the biggest issues is time consuming,non transparency, ineffiency. Using this app offers features like giving the best convenient route for drivers that save time, it offers the citizen all types of information that makes it transparent and it improves the efficiency of the waste management process. It means the waste management mobile app can break all the challenges in the existing system. The aim of this application is to improve the quality of management in Waste collection and impacting positively the environment by reducing the quantity of waste.
